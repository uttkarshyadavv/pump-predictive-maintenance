# Pump Predictive Maintenance

This project develops a machine learning based predictive maintenance system for centrifugal pumps. 
The goal is to detect early signs of failure using vibration, pressure, and flow-rate patterns, enabling 
proactive intervention and reducing unplanned downtime.

---

## Features
- Signal preprocessing and smoothing
- Time-domain and frequency-domain feature extraction
- Anomaly detection pipeline
- Random Forest / XGBoost / LSTM model options
- Visualization of degradation patterns
- Exportable models for deployment
